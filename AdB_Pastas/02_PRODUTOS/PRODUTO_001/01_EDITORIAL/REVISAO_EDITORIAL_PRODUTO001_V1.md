# REVISÃO 01
## Aumentar a percepção de livro

Problema

Em diversos momentos o material ainda transmite a sensação de um documento diagramado.

O objetivo deve ser fazer o leitor sentir que está lendo um livro publicado por uma editora.

Ações

• aumentar respiros

• ampliar margens internas

• utilizar mais páginas

• reduzir sensação de densidade

Nunca tentar economizar páginas.

---

# REVISÃO 02
## Hierarquia tipográfica

Problema

O texto possui excelente organização.

Entretanto quase todos os elementos possuem peso visual semelhante.

Resultado:

o leitor não percebe imediatamente onde está a informação mais importante.

Ações

Aumentar:

Título principal

Subtítulos

Frases-chave

Conceitos

Definições

Criar maior contraste entre:

H1

H2

H3

Texto

Legenda

Checklist

Citação

---

# REVISÃO 03
## Criar identidade para os componentes

Hoje:

Os boxes parecem apenas textos destacados.

Objetivo:

Cada componente deve possuir identidade própria.

Padronizar:

💡 Dica

⚠ Atenção

📖 Estudo de Caso

🎯 Missão

📄 Documento Fundador

🧰 Ferramenta

✓ Checklist

Todos devem possuir:

cor

ícone

espaçamento

tipografia

estilo próprios.

O leitor deve reconhecer o componente antes mesmo de começar a ler.

---

# REVISÃO 04
## Melhorar as figuras

Problema

As imagens hoje ilustram.

Precisam ensinar.

Cada figura deve possuir:

Título

Legenda

Comentário

Quando aplicável:

setas

comparações

destaques

numeração

balões

Exemplo

Errado

Foto da fachada.

Correto

Figura mostrando

fachada

placa

entrada

iluminação

sinalização

com chamadas explicativas.

As figuras devem funcionar mesmo sem leitura do texto.

---

# REVISÃO 05
## Criar páginas de impacto

Todo capítulo deve possuir páginas memoráveis.

Exemplos:

Página inteira apenas com uma frase.

Página apenas com uma fotografia.

Página apenas com um conceito.

Página apenas com uma pergunta.

Exemplo:

MARCA

NÃO É

LOGOTIPO.

Essas páginas diminuem o cansaço de leitura.

---

# REVISÃO 06
## Melhorar o ritmo

Algumas páginas concentram muito texto.

Não reduzir conteúdo.

Distribuir melhor.

Meta:

Nenhuma página deve parecer cansativa.

Preferir

mais páginas

menos densidade.

---

# REVISÃO 07
## Direção de Arte

As imagens precisam parecer parte do livro.

Não elementos adicionados posteriormente.

Padronizar:

tratamento

enquadramento

temperatura

luz

proporção

estilo fotográfico

Todas as imagens devem pertencer ao mesmo universo visual.

---

# REVISÃO 08
## Criar assinatura editorial

Toda publicação da Academia deve possuir elementos recorrentes.

Exemplos:

barra dourada

linha editorial

numeração

rodapé

capítulo

ícones

estilo das legendas

caixas

Isso fortalece identidade.

---

# REVISÃO 09
## Melhorar o Documento Fundador

Este é o maior diferencial do livro.

Transformá-lo em um elemento visual recorrente.

A cada capítulo:

mostrar progresso.

Exemplo

□□□□□□□□

■□□□□□□□

■■□□□□□□

■■■□□□□□

...

O leitor deve sentir evolução.

---

# REVISÃO 10
## Experiência Premium

Inserir pequenas pausas editoriais.

Exemplos

"Pare um minuto."

"Observe sua barbearia."

"Antes de continuar..."

Essas pausas fazem o leitor participar.

---

# REVISÃO 11
## Padronização das fotografias

Todas as fotografias devem seguir um mesmo padrão.

Preferencialmente:

fotografia documental

luz natural

ambientes reais

barbearias brasileiras

clientes reais

evitar aparência de banco de imagens.

---

# REVISÃO 12
## Valor percebido

O livro já possui conteúdo de alto nível.

Agora o objetivo é aumentar a percepção de valor.

O leitor deve pensar:

"Isso parece um livro de R$ 197."

Não um PDF gratuito.
