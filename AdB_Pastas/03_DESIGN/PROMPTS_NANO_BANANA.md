# INSTRUÇÕES DE USO

Este documento contém prompts mestres organizados por categoria.

Cada prompt segue a filosofia e os parâmetros definidos no GUIA_DIRECAO_DE_ARTE_AB.md.

Nenhum prompt deve ser usado sem revisão do Teste Final (6 perguntas).

## Como adaptar

Cada prompt contém variáveis em colchetes: `[VARIÁVEL]`.

Substituir pela especificação do projeto antes de gerar.

Exemplos de variáveis frequentes:

- `[PRODUTO]` → Kit Fundação / Mini Manual / Cartão de Bolso
- `[NÍVEL]` → baixo investimento / médio investimento / alto investimento
- `[FERRAMENTA]` → tesoura / navalha / máquina / pente
- `[MOMENTO]` → manhã / tarde / horário de pico

# NOTAS DE PRODUÇÃO

## Consistência entre imagens

Quando produzir múltiplas imagens para o mesmo produto:

1. Definir e manter o mesmo `[TOM DE LUZ]` em todas as imagens
2. Usar o mesmo `[TIPO DE BARBEIRO]` quando o mesmo personagem reaparecer
3. Manter o mesmo `[AMBIENTE]` quando a continuidade for necessária
4. Revisar todas com o Teste Final antes de aprovar qualquer conjunto

## Ordem recomendada de produção por produto

**Para capas:** 01.A ou 01.B primeiro. Testar em miniatura antes de gerar em alta resolução.

**Para conteúdo interno:** Categorias 04 e 06 primeiro. Definem o tom de todo o material.

**Para Instagram:** Categorias 09 e 04 em paralelo. Garantir coerência visual entre feed e conteúdo educativo.
