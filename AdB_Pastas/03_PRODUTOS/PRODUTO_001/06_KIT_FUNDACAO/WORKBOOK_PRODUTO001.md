# COMO USAR ESTE WORKBOOK

Este Workbook é o par prático do Guia da Identidade da Barbearia.

Cada seção corresponde a um capítulo do guia e contém uma única coisa: **a Missão que você precisa executar**.

Sem teoria. Sem resumos. Sem explicações longas.

Aqui você age.

---

**A sequência correta:**

```
Leia o capítulo no Guia
        ↓
Abra este Workbook
        ↓
Execute a Missão
        ↓
Preencha o Plano de Ação
        ↓
Avance para o próximo capítulo
```

---

**Três regras antes de começar:**

**1. Não avance sem completar a Missão.**
Uma Missão incompleta é um capítulo incompleto. O método só funciona em sequência.

**2. Escreva — não apenas pense.**
A diferença entre uma decisão tomada e uma intenção é o ato de escrever. Se não está no papel, ainda não é uma decisão.

**3. Use o Documento Fundador como destino.**
Cada Missão produz algo que vai direto para o Documento Fundador da Barbearia. Ao terminar este Workbook, o Documento estará preenchido.
