# Objetivo

O Framework FDP-AB é a metodologia proprietária da Academia da Barbearia.

Seu objetivo é orientar o desenvolvimento profissional de barbeiros e empresários do setor por meio de uma jornada simples, prática e aplicável.

Todos os produtos da Academia devem, sempre que possível, estar alinhados a este Framework.

---

# O Problema

Grande parte dos barbeiros domina a técnica do corte, mas nunca recebeu formação estruturada para construir um negócio forte.

Na prática, isso gera barbearias com boa execução técnica, porém sem identidade, diferenciação e consistência.

O Framework FDP-AB foi criado para reduzir essa lacuna.

---

# A Transformação

**Antes**

> "Sou um bom barbeiro."

↓

**Depois**

> "Construí uma barbearia profissional, memorável e preparada para crescer."

---

# Princípios do Framework

O Framework é sustentado por cinco princípios.

### 1. Toda decisão comunica.

Cada detalhe da barbearia influencia a percepção do cliente.

### 2. Marca é percepção.

A identidade de uma empresa é construída pelo que ela entrega, não pelo que ela diz.

### 3. Conhecimento só gera valor quando melhora decisões.

Aprender sem aplicar não gera transformação.

### 4. A experiência deve ser intencional.

Nada importante deve acontecer por acaso.

### 5. Crescimento depende de consistência.

Negócios fortes são construídos por processos repetíveis.

---

# A Jornada FDP

O desenvolvimento profissional acontece em três etapas.

```
FUNDAMENTAR
        ↓
DIFERENCIAR
        ↓
PERPETUAR
```

Cada etapa prepara a próxima.

---

# Etapa 1 — FUNDAMENTAR

**Pergunta principal**

> Quem eu sou como marca?

Nesta etapa o barbeiro constrói as bases do seu negócio.

Objetivos:

* definir propósito;
* definir posicionamento;
* conhecer o público;
* estabelecer valores;
* criar identidade.

Resultado esperado:

Uma direção clara para todas as decisões futuras.

---

# Etapa 2 — DIFERENCIAR

**Pergunta principal**

> Como quero ser percebido?

Nesta etapa a identidade se transforma em experiência.

Objetivos:

* identidade visual;
* ambiente;
* atendimento;
* comunicação;
* presença digital;
* experiência do cliente.

Resultado esperado:

Uma marca percebida como profissional e memorável.

---

# Etapa 3 — PERPETUAR

**Pergunta principal**

> Como manter essa qualidade ao longo do tempo?

Nesta etapa o foco deixa de ser criar e passa a ser manter e evoluir.

Objetivos:

* padronização;
* gestão;
* planejamento;
* melhoria contínua;
* evolução do negócio.

Resultado esperado:

Uma barbearia consistente, organizada e preparada para crescer.

---

# Relação com o Modelo CHA

O Framework FDP-AB define a jornada de desenvolvimento.

O modelo CHA define as competências desenvolvidas durante essa jornada.

Em cada etapa devem ser estimulados:

* Conhecimentos;
* Habilidades;
* Atitudes.

As Competências Digitais atravessam todas as etapas.

---

# Relação com a Metodologia Pedagógica

Todo conteúdo da Academia deve seguir a sequência:

Explicar

↓

Demonstrar

↓

Aplicar

↓

Transformar

Essa metodologia é utilizada para ensinar o Framework FDP-AB.

---

# Aplicação do Framework

O Framework deve orientar todos os produtos da Academia.

Exemplos:

| Produto                      | Etapa Principal |
| ---------------------------- | --------------- |
| Guia da Identidade           | Fundamentar     |
| Guia do Atendimento          | Diferenciar     |
| Templates para Redes Sociais | Diferenciar     |
| Checklists Operacionais      | Perpetuar       |
| Planner da Barbearia         | Perpetuar       |
| Ferramentas de IA            | Perpetuar       |

---

# Como Utilizar

Sempre que um novo conteúdo for criado, deve responder às seguintes perguntas:

1. Em qual etapa do Framework ele atua?
2. Qual problema resolve?
3. Quais competências desenvolve?
4. Como gera aplicação prática?

Se essas perguntas não puderem ser respondidas, o conteúdo provavelmente não está alinhado ao Framework.

---

# O que o Framework não é

O Framework FDP-AB não é:

* um checklist;
* um curso;
* um plano de negócios;
* um processo operacional;
* um conjunto de boas práticas.

Ele é uma metodologia de desenvolvimento profissional.

---

# Evolução

**Fundamentar → Diferenciar → Perpetuar**
