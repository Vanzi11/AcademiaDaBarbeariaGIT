# Manual de Direção de Arte

## Academia da Barbearia

Versão 1.0

---

# Finalidade

Este documento estabelece a direção de arte oficial da Academia da Barbearia.

Seu objetivo é garantir que toda fotografia, ilustração, renderização, imagem gerada por Inteligência Artificial, vídeo, material gráfico ou conteúdo digital mantenha uma identidade visual consistente.

Este guia deverá ser seguido por:

* designers;
* fotógrafos;
* videomakers;
* ilustradores;
* diagramadores;
* social media;
* agências;
* ferramentas de IA.

Nenhuma produção visual deverá contrariar este documento.

---

# Princípio Fundamental

A Academia da Barbearia não vende luxo.

A Academia vende transformação.

Toda imagem produzida deve fazer o barbeiro pensar:

> **"Eu consigo fazer isso na minha barbearia."**

Se uma imagem transmitir uma realidade distante do público, ela deve ser descartada.

---

# Filosofia Visual

Toda comunicação visual da Academia deve transmitir:

✓ profissionalismo

✓ organização

✓ simplicidade

✓ confiança

✓ acolhimento

✓ limpeza

✓ autenticidade

✓ realidade brasileira

Nunca transmitir:

✗ ostentação

✗ excesso de luxo

✗ cenários inalcançáveis

✗ estética de propaganda

✗ exagero

✗ artificialidade

---

# A Barbearia Brasileira

Toda produção visual deverá representar prioritariamente a realidade da maior parte das barbearias brasileiras.

Perfil predominante:

* bairro;
* pequeno porte;
* 1 ou 2 cadeiras;
* espaço reduzido;
* orçamento limitado;
* ambiente simples;
* bem organizado;
* limpo;
* profissional.

A Academia inspira evolução.

Nunca frustração.

---

# O Público

As imagens devem representar pessoas reais.

Nunca modelos publicitários.

Diversidade é obrigatória.

Sempre representar diferentes:

* idades;
* tons de pele;
* estilos;
* profissões;
* regiões do Brasil.

---

# Perfil do Barbeiro

Idade aproximada:

25–50 anos.

Visual:

* natural;
* aparência comum;
* barba opcional;
* tatuagens discretas quando fizer sentido;
* uniforme simples;
* expressão acolhedora.

Nunca parecer modelo de catálogo.

Nunca exagerar na estética "hipster americana".

---

# Perfil do Cliente

Representar clientes reais.

Exemplos:

* trabalhador;
* estudante;
* pai de família;
* adolescente;
* criança;
* idoso;
* profissional liberal.

Sempre transmitir identificação.

---

# Estilo Fotográfico

Priorizar:

Fotografia documental.

A sensação deve ser:

"Isso aconteceu de verdade."

Nunca parecer:

campanha publicitária.

---

# Linguagem Visual

A imagem deve ensinar.

Nunca decorar.

Cada fotografia precisa comunicar uma ideia.

Cada imagem deve responder:

"O que o barbeiro aprende olhando para isso?"

# Iluminação

Priorizar:

luz natural.

Quando artificial:

temperatura quente.

Nunca utilizar:

HDR exagerado.

Contrastes excessivos.

Neons coloridos.

Filtros pesados.

---

# Paleta de Cores

Base:

preto

cinza

branco

madeira

tons terrosos

verde discreto

azul discreto

Evitar:

cores extremamente saturadas.

---

# Materiais

Priorizar:

madeira.

metal.

vidro.

cimento.

tecidos.

couro.

materiais reais.

Evitar cenários excessivamente sofisticados.

---

# Organização

Toda barbearia deve parecer:

organizada.

limpa.

funcional.

Mesmo quando simples.

Nunca transmitir bagunça.

---

# Expressões

Cliente:

relaxado.

confiante.

acolhido.

Barbeiro:

atencioso.

concentrado.

seguro.

Nunca poses artificiais.

# O que Nunca Mostrar

Nunca utilizar:

* Ferraris;
* motos de luxo;
* Harley Davidson;
* charutos;
* whisky;
* barbearias gigantes;
* ambientes irreais;

---

# Direção para IA

Sempre que produzir uma imagem utilizando Inteligência Artificial:

Descrever:

* ambiente;
* iluminação;
* enquadramento;
* expressão;
* materiais;
* textura;
* organização;
* horário;
* clima;
* objetivo pedagógico.

Nunca utilizar prompts genéricos.

A IA deve receber instruções equivalentes às de um fotógrafo profissional.

---

# Princípio Editorial

Toda imagem produzida pela Academia deve cumprir pelo menos uma destas funções:

✓ ensinar;

✓ comparar;

✓ demonstrar;

✓ inspirar;

✓ orientar;

✓ facilitar uma decisão.

Se a imagem apenas decorar a página, ela não deve ser utilizada.

---

# Teste Final

Antes de aprovar qualquer imagem, responder:

Esta imagem parece brasileira?

O barbeiro consegue se identificar?

Ela ensina alguma coisa?

Ela transmite profissionalismo?

Ela parece possível de reproduzir?

Ela conversa com o restante da Academia?

Se qualquer resposta for negativa, a imagem deverá ser refeita.

---

# A Regra de Ouro

A Academia da Barbearia não mostra a barbearia dos sonhos.

Mostra a próxima evolução possível.

É isso que gera identificação.

É isso que gera confiança.

É isso que transforma.

E é isso que fará nossos materiais permanecerem relevantes por muitos anos.
